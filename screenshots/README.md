# Screenshots Folder

วางภาพหน้าจอที่นี่ (optional):

- `workflow-overview.png` - ภาพรวมของ Workflow
- `workflow-result.png` - ผลการทดสอบ Workflow
- `test-result.png` - ผลการรัน test

## วิธีการ Screenshot

### Windows
- กด `Win + Shift + S` แล้วเลือกพื้นที่
- หรือใช้ Snipping Tool

### Mac
- กด `Cmd + Shift + 4` แล้วเลือกพื้นที่

### n8n
- คลิกขวาที่ Canvas > Save as image
